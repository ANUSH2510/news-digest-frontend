# AI News Digest - Frontend

A React web app for personalized AI-summarized news digest.

## Features
- Signup and Login with JWT
- Topic selection
- Personalized news feed with AI summaries
- PWA support for mobile install
- Responsive design

## Tech Stack
- React 18
- Vite
- React Router DOM
- Axios

## Setup
1. Clone the repo
2. `npm install`
3. `npm run dev`

## Deployment
Frontend deployed on Vercel.
